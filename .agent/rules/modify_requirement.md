## UI 设计与功能优化需求

### 1. 设计系统与规范 (Design System & Guidelines)
- **核心风格**: 严格遵循 [Material You] 设计准则（如大圆角、动态主题色彩搭配、高度差阴影与扁平化结合）。
- **专业级 UI/UX (遵循 `ui-ux-pro-max` Skill)**:
  - **图标使用**: 仅使用标准的 SVG 图标（推荐 Material Symbols / Lucide 等），**严禁用 Emoji 充当 UI 元素**。
  - **交互反馈**: 所有可点击的卡片、按钮需带有 `cursor-pointer` 及明确的悬停 (Hover) / 焦点 (Focus) 变色与过渡动画（如 `transition-colors duration-200`）。
  - **色彩与模式**: 必须兼容 Light/Dark 模式，确保卡片背景对比度、文字可读性和边框的可见性。
  - **布局层级**: 浮动组件（如 Navbar、弹窗）需保留足够的安全边距，避免遮挡底层核心内容。

### 2. 用户主页重构 (User Dashboard)
- **核心定位 - 进度追踪**: 主页的首要功能必须是**已标记条目**（如追剧、追番进度）的独立网格或列表展示面板，而非一个空白的纯搜索页。
- **风格重绘**: 参考根目录下 `@bangumi_track.img` 的直观排版理念（分进度条、海报、状态等信息传递），但**必须将其彻底改造为 Material You 风格**的现代化设计。

### 3. 搜索交互 (Search Experience)
- **空间优化**: 搜索功能不应常驻并霸占主页视觉核心。
- **动态展开**: 默认状态下，主页右上角（App Bar 内）只保留一个“搜索图标按钮（Icon Button）”。当用户点击时，再通过动画平滑展开为完整的搜索输入框（常见于展开式搜索或 Searchbar 浮层交互）。
